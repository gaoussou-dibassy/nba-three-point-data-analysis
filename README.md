# NBA Three Point Analysis 2025-26

This project analyzes NBA team three point efficiency for the 2025-26 regular season. It looks at which teams are the most efficient from three, how volume factors into overall three point value, and how the New York Knicks compare to the rest of the league.

## Libraries Used
- nba_api
- pandas
- matplotlib
- numpy

## How to Run
Install dependencies:
pip install nba_api pandas matplotlib numpy

Then open and run nba-three-point-analysis.ipynb in Jupyter.
